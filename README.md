# Data Structures and Algorithms (DSA) Solutions

This repository contains a comprehensive collection of Data Structures and Algorithms problems solved across various platforms like LeetCode, GeeksforGeeks, CodeStudio, and more. The solutions are organized into folders based on their specific topics or problem types to make navigation and learning easier.

## 📂 Folder Structure

Each folder represents a specific category or data structure. Inside each folder, you will find the source code for various problems related to that topic.

- **Arrays**: Problems related to 1D and 2D arrays, sliding windows, and two-pointer techniques.
- **Linked Lists**: Implementation and manipulation of Singly, Doubly, and Circular Linked Lists.
- **Stacks & Queues**: Problems involving LIFO and FIFO structures, including monotonic stacks.
- **Trees**: Binary Trees, Binary Search Trees (BST), and AVL trees.
- **Graphs**: BFS, DFS, Dijkstra’s, Minimum Spanning Trees, and Topological Sorting.
- **Dynamic Programming**: Optimization problems solved using memoization and tabulation.
- **Recursion & Backtracking**: Classic problems like N-Queens, Sudoku Solver, and Permutations.
- **Searching & Sorting**: Binary Search, Quick Sort, Merge Sort, and their variations.
- **Strings**: Pattern matching, anagrams, and string manipulation.
- **Heaps**: Priority Queue implementations and K-th element problems.

## 🛠️ How to Use

1. Navigate to the folder of the topic you wish to study.
2. Open the specific problem file (usually named after the problem title).
3. Each file contains the optimized solution along with comments explaining the time and space complexity.

The repository focuses on providing clean, from-scratch implementations of data structures and algorithms. Each solution is designed to be self-contained, demonstrating the core logic without relying on external libraries where possible.
